1, 基本的なウェブサイトの作り方

　この世のウェブサイトは、だいたいが「html」という言語で書かれています。基本的には以下のような構造でできています。

<!DOCTYPE html>

<html lang="ja">

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>※ここにページのタイトル</title>

</head>

<body>

    ※ここにウェブサイトの内容

</body>

</html>

<三角かっこ>は「タグ」とよばれており、htmlはタグを羅列して書いていきます。<html>・・・</html>のように、間に文字列(これをコンテンツと呼ぶ)をはさんで使うものもあります。このとき、<html>の方を開始タグ、</html>の方を終了タグと呼びます。

　タグの基本的な書き方は以下の通りです。

<タグの名前 属性名=値>コンテンツ</タグの名前>

例外もありますが、大半のタグはこのように書きます。「属性」というのを追加でき、色などいといろなことを指定できます。省略も可能です。

　そして、cssを適応する場合、<head></head>の間に<link rel="stylesheet" href="ファイル名">と書いてcssとhtmlを紐づけて、適応したい要素に

<a class="クラスの名前">ボタンも作れるよ</a>

というようにクラスの名前を書いてあげることで適応できます。

クラスの名前に使える文字はアルファベット、数字、-(ハイフン)、_(アンダースコア)だけです。

　cssの適応の方法は他にもありますが、とりあえずはこれで大丈夫だと思います。

 次にcssの中身の解説をします。まずは前半部分。

a.btn--blue {

    color: #fff;

    padding: 20px;

    background-color: #0000ff;

  }

まず、color: #fff;の部分。文字の色を指定しています。「#fff」のようにRGB値で指定します。これは知っている人も多いと思います。RGB値とは、Red、Green、Blueという光の3原色をどの程度ずつ混ぜるかを指定することで色を指定できるものです。16進数でしていし、1文字目は赤の量、2文字目は緑の量、3文字目は青の量です。今回は全てfなので、白となります。

　次にpadding: 20px;の部分。Paddingとは余白のことで、余白の大きさを指定できます。単位のpx(ピクセル)は画面の最小単位のつぶつぶのことです。余白はもっと細かく指定もできますが、このコーナーは初心者向けなので、気になる人は自分でググってください。

　最後にbackground-color: #0000ff;の部分。背景の色、つまりボタンの色を指定できます。ここでもRGB値を使いますが、先ほどのとは桁数が違いますね。実はRGB値は6桁(つまり1つの原色あたり2文字)でも指定できます。こちらも16進数です。今回は3つめの原色、つまり青をマックスのffにして、他の原色をすべて00にしているため、青になります。

まずは、htmlの方から解説していきます。

<p id="kirikae">表示される</p>

<input type="button" value="ボタン" onclick="clickBtn()" />

<p id=”ID名”></p>というように要素にIDをつけてあげます。これでjavascriptの方から制御できるようになります。

そしてボタンは<input/>を使います。typeはbutton(ボタン)。Valueでボタンに表示する文字列を指定します。そしてonclickでは、押したときに実行するjavascriptの関数を指定します。関数とは命令をひとまとめにしたセットのようなものです。

　これでhtml側での記述は終わりました。次にjavascript側です。まず、

document.getElementById("kirikae").style.visibility ="hidden";

でidが”kirikae”の要素のvisibility(可視性)を”hidden”にする(=つまり隠す)という命令です。Javascriptの命令はまじでたくさんあるので、調べながら書くことが多いです。

　このしたで関数(function)を書きます。

function 関数名(){

　　　　※ここに関数の内容

    }

関数の名前はアルファベットです。今回はclickBtnと名付けました。

関数の中身を解説していきましょう。

const kirikae = document.getElementById("kirikae");

これで定数kirikaeにdocument.getElementById("kirikae")が代入されました。これでどうなるかというと、これ以降では

document.getElementById("kirikae").style.visibility ="hidden";

ではなく

kirikae.style.visibility ="hidden";

でよくなるということです。これで記述が楽になります。

次に

if(kirikae.style.visibility=="visible"){

        kirikae.style.visibility ="hidden";

    }

    else{

        kirikae.style.visibility ="visible";

    }

まず条件分岐について解説する必要があります

    if(条件){

        ※命令;

    }

    else{

        ※命令;

    }

    }

こういう感じです。ifは「もし～なら」という英単語ですね。elseは「もし～でなければ」という意味で用いられます。ここではkirikae.style.visibility=="visible"が条件になっており、kirikae(先ほど代入した定数。ここではIdがkirikaeの要素のこと。)のvisibility(可視性)が”visible”(見える状態)だったら実行するということです。==はイコールです。条件の場合2つ続けて書きます。条件の書き方もたくさんありますが、説明しているときりがないので気になる人はググってください。

今回の条件分岐を日本語に訳すなら、「もしkirikaeが見えるなら見えなくする、そうでないなら見えるようにする。」ということです。簡単ですね。
